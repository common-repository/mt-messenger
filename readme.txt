Add Facebook's Messenger live Chat head/box to your site for Facebook page

== Description ==
This plugin uses Facebook's Official JS SDK to display your Facebook page chat on  your website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1.  Install this plugin by using WordPress built in "add new plugin" feature or pasting the plugin folder in /wp-content/plugins folder
2.  Make an page or go to existing page settings
3.  navigate to messenger platform
4.  add your website to white-listed domains or you can also do this more efficiently by clicking Setup Chat plugin on page
5.  follow the wizard, when wizard prompt code Snippet copy the value of "page_id='123'", in here 123 is your id; Note that your id is always numeric
6.  Activate and open the plugin page, you will see one single input field paste your page id and click submit
7.  Refresh your site page to see Messenger chat box added on your right bottom side of page
